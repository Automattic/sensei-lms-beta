Easily update to prerelease versions of Sensei LMS for testing and development
purposes.

== Description ==

**Sensei LMS Beta Tester** allows you to try out new versions of Sensei LMS
before they are officially released.

**Use with caution, not on production sites. Beta releases may not be stable.**

After activation, you'll be able to choose an update channel:

1. Beta - Update to beta releases, RC, or stable, depending on what is newest.
2. Release Candidate - Update to RC releases or stable, depending on what is
newest.
3. Stable - No beta updates. Default WordPress behavior.

== Frequently Asked Questions ==

= Does this allow me to install multiple versions of Sensei LMS at the same
time?

No; updates will replace your currently installed version of Sensei LMS. You
can switch to any version from this plugin via the interface however.

= Where do updates come from? =

Updates are downloaded from the GitHub repository where we push betas and release candidates.

= Where can I report bugs or contribute to Sensei LMS Beta Tester? =

Bugs can be reported to the [Sensei LMS Beta Tester GitHub issue tracker](https://github.com/Automattic/sensei-lms-beta).

= Where can I report bugs or contribute to Sensei LMS? =

Join in on our [GitHub repository](https://github.com/Automattic/sensei/).
